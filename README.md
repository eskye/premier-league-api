### Installation

Follow these steps to set up the app.

#### Clone the repo:

`$ git clone https://github.com/igbominadeveloper/premier-league-api.git`

#### Navigate to the project directory:

`$ cd premier-league-api`

#### Install dependencies

`npm install`

#### Start the application

`npm run start:dev`
